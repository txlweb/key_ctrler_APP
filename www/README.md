# 侧键控制器官网

这是侧键控制器应用的官方静态网站。

## 文件结构

- `index.html` - 主页面，包含应用介绍、使用说明和下载途径

## 特性

- 🎨 **一加红主题设计** - 采用一加品牌的经典红色配色方案
- 📱 **响应式布局** - 适配桌面和移动设备
- ⚡ **现代化UI** - 使用渐变、阴影和动画效果
- 🔗 **QQ群集成** - 直接链接到官方交流群

## 本地预览

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   ```

## 部署

将整个 `www` 文件夹上传到任何静态网站托管服务即可，如：
- GitHub Pages
- Netlify
- Vercel
- 阿里云OSS
- 腾讯云COS
